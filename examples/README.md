# Examples

This directory contains example configurations and usage guides for FlagScale-Agent.

## Configuration Examples

### Agent Configuration
- [`agent_config.yaml`](agent_config.yaml) — Complete agent configuration with all options

### Training Workflow Examples
- [`train_qwen2_7b.md`](train_qwen2_7b.md) — End-to-end Qwen2.5 7B training workflow
- [`diagnose_training.md`](diagnose_training.md) — Debugging a training failure

## Running Examples

1. Ensure FlagScale-Agent is installed:
   ```bash
   pip install flagscale-agent
   ```

2. Set your API key:
   ```bash
   export ANTHROPIC_API_KEY="your_key_here"
   ```

3. Start the agent and paste the example queries.
